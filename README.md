# Predictive Maintenance using Neural Networks

## 📌 Overview
This project aims to develop a neural network-based predictive maintenance system for smart factories. The system analyzes multi-sensor data to predict potential equipment failures in advance, helping reduce downtime and maintenance costs.

## 🎯 Objectives
- Predict machine failures using sensor data
- Reduce false alarms
- Improve system efficiency
- Enable real-time maintenance decisions

## 🧠 Approach
- Exploratory Data Analysis (EDA)
- Feature Engineering & Fusion
- Anomaly Detection (Autoencoder)
- Deep Learning Model (LSTM/GRU with Attention)
- Model Evaluation & Optimization
- Explainable AI (SHAP / Attention Visualization)
- Deployment using Flask & Streamlit Dashboard

## 📊 Dataset
AI4I 2020 Predictive Maintenance Dataset
